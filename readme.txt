This is a WordPress plugin that displays related posts under the content of a post to improve user engagement.

== Description ==

The RA Related Posts plugin enhances user engagement by displaying related posts at the end of the content of each post. The related posts are fetched based on the categories of the current post. This plugin helps improve navigation within your website by offering readers additional relevant content.

== Theme Used ==

Twenty Twenty Four

**Features:**

* Displays a maximum of 5 related posts based on the current postâ€™s categories.
* Each related post includes a thumbnail image, author name, title, and published date.
* The related posts are displayed in a responsive grid layout.
* Uses a custom CSS file (`custom.css`) to style the related posts section.

== Installation ==

1. Upload the `ra-related-posts` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The related posts will automatically appear at the end of each post's content.

== Frequently Asked Questions ==

= How are the related posts determined? =
The related posts are fetched based on the categories of the current post. The plugin randomly selects up to 5 posts from the same categories.

= Can I customize the appearance of the related posts? =
Yes, you can customize the appearance by editing the `css/custom.css` file located within the plugin folder.

= Does this plugin work with custom post types? =
This version of the plugin is designed to work with the default 'post' post type. Future versions may include support for custom post types.
